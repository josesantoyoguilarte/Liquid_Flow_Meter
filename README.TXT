

PROJECT TITLE: Liquid Flow meter.

PURPOSE OF PROJECT: To fulfill with the requirements of an engineering course SYSC 3010  and to exert expertise in computer systems and team work.

VERSION or DATE: 1.0

HOW TO START THIS PROJECT:For this project it is required the use of two Raspberry Pi. Each of one to be connected to a monitor via HDMI cable. In one Raspberry Pi the python program is ran, and in the other Raspberry Pi the java program is ran. 
Note: The two Raspberry Pi must be physically connected over the same LAN
